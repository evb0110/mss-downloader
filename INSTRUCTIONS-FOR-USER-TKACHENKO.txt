================================================================================
ШАГ 1: СКАЧАЙТЕ АВАРИЙНЫЙ СКРИПТ
================================================================================

Откройте эту ссылку в браузере (скопируйте целиком):
https://raw.githubusercontent.com/evb0110/mss-downloader/main/EMERGENCY-FIX-RUN-THIS.bat

Сохраните файл на Рабочий стол как: EMERGENCY-FIX-RUN-THIS.bat

================================================================================
ШАГ 2: ЗАПУСТИТЕ СКРИПТ
================================================================================

1. Найдите файл EMERGENCY-FIX-RUN-THIS.bat на Рабочем столе
2. Двойной клик для запуска
3. Если спросит Windows - нажмите "Запустить всё равно"

================================================================================
ШАГ 3: ЧТО ДЕЛАЕТ СКРИПТ
================================================================================

Скрипт автоматически:
✓ Убивает ВСЕ старые процессы MSS-Downloader
✓ Очищает все файлы блокировок
✓ Находит установленную программу
✓ Запускает её с диагностикой
✓ Показывает ошибки если они есть

================================================================================
ЕСЛИ НЕ ПОМОГЛО - РУЧНОЙ ЗАПУСК
================================================================================

1. Откройте Диспетчер задач (Ctrl+Shift+Esc)
2. Вкладка "Подробности"
3. Найдите и завершите ВСЕ процессы:
   - Abba Ababus (MSS Downloader).exe
   - MSS-Downloader.exe
   - mss-downloader.exe

4. Найдите программу:
   C:\Users\[ВашеИмя]\AppData\Local\Programs\abba-ababus-mss-downloader\

5. Правый клик на "Abba Ababus (MSS Downloader).exe"
6. Свойства → Совместимость
7. Поставьте галочки:
   ✓ Запускать программу в режиме совместимости: Windows 8
   ✓ Выполнять эту программу от имени администратора
8. Нажмите OK и запустите
