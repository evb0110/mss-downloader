ИСПРАВЛЕНИЕ ISSUE #5 - ФЛОРЕНЦИЯ (Florence) ETIMEDOUT
=====================================================

ПРОБЛЕМА:
--------
Библиотека Флоренции (cdm21059.contentdm.oclc.org) выдавала ошибку ETIMEDOUT при попытке загрузки манускриптов. 
Сервер по адресу 193.240.184.109:443 не отвечал на запросы в установленные временные рамки.

URL для тестирования: https://cdm21059.contentdm.oclc.org/digital/collection/plutei/id/317515/

РЕШЕНИЕ:
--------
1. Добавлена поддержка Florence в метод fetchWithHTTPS для улучшенной обработки соединений
2. Реализована логика повторных попыток при ETIMEDOUT с экспоненциальной задержкой
3. Добавлено предварительное разрешение DNS для избежания таймаутов при резолве домена
4. Настроен пул соединений с keep-alive для улучшения производительности
5. Добавлены специфичные для Florence сообщения об ошибках
